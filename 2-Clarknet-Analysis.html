One immediately see some of requests with a lot of "Internal Server Error" response codes. Now it is time to investigate those systematically.

This is what I am going to do for each request step by step.

  • mark requests with codes 403, 404 or 505 as "bad" via binary variable

  • for each request count "bad" and "total" response_codes

  • divide "bad" counts by the "total" ones to get the percent of "bad"

The results are sorted by percentage of "bad" and then by counts of "bad" in descending order

In [20]:
percentage_bad_DF = (GETlogs_DF.select("request","response_code")
                               .withColumn("is_bad", 
                                           func.when(GETlogs_DF.response_code.isin(403, 404, 505), 1).otherwise(0))
                               .groupBy("request")
                               .agg(func.sum("is_bad").alias("bad"), func.count("is_bad").alias("total"))  
                               .withColumn("percent_bad", func.col("bad")/func.col("total"))
                               .sort("percent_bad", "total", ascending=False)
                               )
percentage_bad_DF.show(5)
